# Time Wallet

**Time Wallet** は、活動時間をタグで記録し、円グラフ・カレンダー・分析画面で振り返るための時間管理Webアプリです。

家計簿が「お金の使い道」を見える化するように、Time Wallet は「時間の使い道」を見える化します。日々の授業、課題、研究、睡眠、趣味などをタグごとに記録し、自分の時間の使い方をあとから振り返れるようにすることを目的にしています。

## 概要

Time Wallet は「時間家計簿」として使える個人向けの時間管理アプリです。

タイマーで現在の活動を記録したり、あとから手動で記録を追加したりできます。保存した記録は、記録一覧・カレンダー・分析画面で確認でき、タグ別の時間配分や日ごとの傾向を見直せます。

未ログイン時はブラウザの localStorage に保存し、ログイン中は Supabase にクラウド保存します。未ログイン中に端末へ保存したデータは、ログイン後に現在のアカウントへ取り込めるようにしています。

スマホで日常的に使うことを想定し、PWA としてホーム画面に追加してアプリのように起動できる構成にしています。

## 主な機能

- タイマー記録
- 画面遷移・リロード後も復元できる記録中タイマー
- 手動記録追加
- 記録一覧
- 記録編集・削除
- タグ管理
- タグ名・タグ色の編集
- カレンダー表示
- 分析画面

- 現在の保存状態表示
- PWA対応

## 保存方式

| 状態 | 保存先 | 内容 |
| --- | --- | --- |
| 未ログイン | localStorage | この端末・このブラウザにタグ、記録、記録中タイマーを保存 |
| ログイン中 | Supabase | ログイン中のユーザーに紐づけてタグ、記録、記録中タイマーをクラウド保存 |

未ログイン中に localStorage へ保存したタグ・記録は、ログイン後に「この端末のデータを取り込む」操作で Supabase 側へ移行できます。

取り込み時は localStorage データを自動削除せず、`legacy_local_id` を使って重複登録を避ける方針にしています。

## Supabase テーブル

Time Wallet では以下のテーブルを使用します。

| テーブル | 役割 |
| --- | --- |
| `profiles` | ユーザーの表示名やタイムゾーンなど、ユーザープロフィール情報を管理 |
| `activity_tags` | 活動タグの名前、色、並び順、有効状態を管理 |
| `time_records` | 開始時刻、終了時刻、記録時間、タグのスナップショットを管理 |
| `active_timers` | 記録中タイマーのタグと開始時刻を管理 |

各テーブルは Supabase Auth のユーザーIDに紐づけて管理し、RLS により自分のデータだけを扱う前提です。

## 環境変数

Supabase 連携には以下の環境変数を使用します。

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

`service_role` key や secret key はクライアント側では使用しません。実際の値は `.env.local` に設定し、GitHub にはコミットしないでください。

## ローカル起動方法

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで以下を開きます。

```txt
http://localhost:3000
```

本番ビルドを確認します。

```bash
npm run build
```

## 開発メモ

このアプリは、時間管理を「記録して終わり」ではなく、「あとから見直して生活を調整する」ためのツールとして開発しています。

家計簿のように、時間もカテゴリごとに眺めることで、自分が何にどれだけ時間を使っているのかを直感的に把握できる体験を目指しています。
